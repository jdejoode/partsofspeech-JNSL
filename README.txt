This repository contains the code and data for a paper by Johan de Joode
on the automated extraction of literary genres based on distributions of 
parts of speech.

The data is taken from the ETCBC.

The script experiment.R is set to be run from within RStudio.

Author: Johan de Joode

Please cite the actual paper in the Journal for North-West Semitic Languages when
using this repo.
